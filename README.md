# Projet Vue.js

Faire un visuel existant UberEats (pas fini)

## Démarrer

Le projet a été créé sous IntelliJ, pour le démarrer suivre les étapes suivantes.

- Cloner le projet

Dans la racine du projet faire :
```bash
npm i
```

```bash
npm run serv
```
